Title: [[ Algebra Linear : Language c ]]
Description: (2003/10/15 version)(use fractions)
 a) Basic matrices functions :
 + - *, power, transpose, trace, determinant, minor, matrix of minor, cofactor, matrix of cofactor, adjoint, inverse, gauss, gaussjordan, linear transformation, LU decomposition , Gram-Schmidt process, similarity.
 b) Basic vectors functions :
 norm, distance, innerproduct,coldim, rowdim, rank, nullity.
* For this work, I use Dev-C++ 4, and DJGPP two freewares. 
* You can also use Ch 4.0 released (C99). The Standard Edition is free in all platforms. 
* There are 30 others package which work with these set of functions.
* All is free, see http://groups.yahoo.com/group/mathc/
